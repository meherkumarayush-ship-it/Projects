# Student Performance Predictor

A machine learning web application that predicts student performance based on academic and personal factors.  
Built with **Python, Flask, HTML/CSS/JS**, and integrated with a trained ML model.

---

## Overview
This project helps analyze student data and predict performance outcomes using machine learning.  
It is designed to be **user-friendly, interactive, and interview-ready**, showcasing both frontend and backend integration.

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask, Python  
- **ML Models:** CNN, Scikit-learn  
- **Deployment:** Flask server  
- **Version Control:** Git & GitHub  


---

## Features
- Predicts student performance based on input factors (attendance, study hours, etc.)  
- Interactive frontend with **form submission without page reload**  
- Backend integration with Flask for seamless data flow  
- Clean UI for better user experience  
- Modular codebase for easy debugging and extension
